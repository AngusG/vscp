int doCmdEnterReceiveLoop ( void  )

This command sets an open interface in the receive loop (RCVLOOP). It does nothing other then putting the interface in the loop mode and checking that it went there.

Note! The only way to terminate this receive loop is to close the session with doCmdClose (or just close the socket).

Returns:
CANAL_ERROR_SUCCESS if success CANAL_ERROR_GENERIC if not.

int doCmdReceiveLevel1 ( canalMsg *  pCanalMsg)

Receive an VSCP Level I event through the interface.

For the extended and the RTR bit to be handled the CAN message bit in the head byte must be set.

Low eight bits of the CAN id is fetched from GUID[15] that is LSB of GUID.

Returns:
CANAL_ERROR_SUCCESS if success false if not.
