Field Documentation

bool m_bFilterOwnTx

Filter our own tx events identified by obid set in m_txChannelID

bool m_bUseRXTXEvents

Send wxWidgets receive event instead of writing event to RX queue. m_pWnd must point to a valid window and the id for that window must be in m_wndID Also TX events will be activated if thread is started.

wxWindow* m_pWnd

Pointer to window that receive events. NULL if not used.

uint32_t m_rxChannelID

Channel id for receive channel

uint32_t m_txChannelID

Channel id for transmit channel If bFilterOwnTx is true events with this obid will not be added to the queue.
