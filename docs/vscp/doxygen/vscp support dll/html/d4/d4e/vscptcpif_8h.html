Detailed Description

The VscpTcpIf class encapuslate sthe VSCP tcp/ip interface.

The VscpTcpIf class holds functionality to talk toi the tcp/ip of the VSCP daemon and on client that implement the interface. All api's of the interface are supported by the class.
