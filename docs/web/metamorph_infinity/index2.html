Welcome to the VSCP Website!

 

 

This site is dedicated to the VSCP protocol, a very simple and free protocol for SOHO and M2M automation tasks & friends. The protocol is so easy that everyone can grasp the idea behind it in a few minutes. Therefore it's also very easy to construct and use VSCP aware modules and components. Nodes just start up and then serve the control solution reliable and stable for ever and ever and ever.....

The protocol is free and is not designed to conquer the world and every Small Office and HOme. It is constructed just because its fun to make tiny little things work together..

Friends of VSCP

 

CANAL - CAN Abstraction Layer. 
A very simple software abstraction layer to CAN drivers. The VSCP daemon and some other applications is now available in its first version for the Windows/Linux platform.

 

OHAS
The Open Home Automation server. A LAMP/WAMP based automation solution.

 

O'DIN  
This is a vision for a Linux based system for higher level SOHO automation. Equipment that just do there work without the fuzz. Install and go.
